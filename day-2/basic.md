### **tokens**:token is the smallest unit of the code, there are 5 types of tokensthey are1.KeywordsReserved words with special meanings.ex: if, else, for, while, def, class2.IdentifiersNames used for variables, functions, classes, etc.3.LiteralsFixed values.4.OperatorsSymbols that perform operations.5.Delimiters (Separators/Punctuators)Symbols that separate program elements.### statements:A statement is an instruction that the Python interpreter can execute.### comments:comments is data written to understand or give briefing about the codecomments can be given two types:1.single line comment&#x20;  ex:#singleline2.multiple line comment&#x20;  ex:'''multiple&#x20;        line'''### variable rules:for defining a variable in python we have 4 conditions they are1. variable can contain A-Z,a-z,0-9,\_2. variable can start with A-Z,a-z,\_3. no keywords are allowed as variables4. **these are case sensitive**#### VARIABLE ASSIGNMENT:for single value a=10same value for diff var a=b=10diff values diff var in single line a,b,c=10,20,30swapping 2 variables a,b=b,a#### PYTHON DATA types:to know the data type; **type()**-> output will be **<class'data type'>****1.numerical:** all are immutable- you can't change it within the same memory reference or object refrence&#x20;    int&#x20;    float&#x20;    complex (j)-imaginary value**2.sequntial:**&#x20;     string; declared using "" or''-immutable \& orderd&#x20;     list; declared using\[],duplicates are allowed, mutable \& orderd&#x20;     tuple; declared using(),immutable, orderd\& duplicates are allowed**3.Mapping :**&#x20;     set; declared using {}, inorderd, duplicates are not allowed, mutable ,to define empty set;s=set()&#x20;     dict: mutable,orderd**4.boolen: True,False**&#x20;**5.none: to declare an unkown data type later you can change to any other data type**&#x20;
