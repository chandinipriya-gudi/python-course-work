##### **Operators in Python**

Python operators are special symbols used to perform operations on variables and values.

**1. Arithmetic Operators**

Used to perform mathematical operations such as addition, subtraction, multiplication, and division.

Operators: +, -, \*, /, %, \*\*, //



**2. Comparison (Relational) Operators**

Used to compare two values and return True or False.

Operators: ==, !=, >, <, >=, <=



**3. Assignment Operators**

Used to assign values to variables.

Operators: =, +=, -=, \*=, /=, %=, //=, \*\*=



**4. Logical Operators**

Used to combine conditional statements.

Operators: and, or, not



**5. Bitwise Operators**

Used to perform operations on binary numbers.

Operators: \&, |, ^, \~, <<, >>



**6. Membership Operators**

Used to check whether a value exists in a sequence.

Operators: in, not in



**7. Identity Operators**

Used to compare the memory locations of two objects.

Operators: is, is not
