#### **Features of Python**

1. Simple and Easy to Learn – Python has a clear and readable syntax, making it beginner-friendly.
2. High-Level Language – It hides complex system details and allows programmers to focus on problem-solving.
3. Interpreted Language – Code is executed line by line without compilation.
4. Object-Oriented – Supports classes, objects, inheritance, polymorphism, etc.
5. Platform Independent – Python code can run on Windows, Linux, and macOS with little or no modification.
6. Dynamically Typed – No need to declare variable types explicitly.
7. Free and Open Source – Available for anyone to use and modify.
8. Large Standard Library – Provides many built-in modules and functions.
9. Portable – Programs can run on different platforms.
10. Extensible – Can be integrated with languages like C and C++.
11. Embeddable – Python code can be embedded into other applications.
12. Supports Multiple Programming Paradigms – Object-oriented, procedural, and functional programming.
13. Automatic Memory Management – Uses garbage collection to manage memory automatically.
14. Robust Exception Handling – Helps handle runtime errors efficiently.
15. Large Community Support – Extensive documentation and community assistance.
16. Suitable for Rapid Development – Faster coding and testing compared to many languages.

##### **OOP (Object-Oriented Programming) Concepts** 

**1. Class**

&#x20;    A class is a blueprint or template used to create objects. It defines the attributes and behaviors that objects will have.



**2. Object**

&#x20;   An object is an instance of a class. It represents a real-world entity with its own data and behavior.



**3. Encapsulation**

&#x20;   Encapsulation is the process of combining data and methods into a single unit (class) and restricting direct access to the data.

Advantages:

Data security

Better control over data

Improved maintainability



**4. Abstraction**

&#x20;   Abstraction is the process of hiding implementation details and showing only the essential features to the user.

Advantages:

Reduces complexity

Enhances security

Makes programs easier to use



**5. Inheritance**

&#x20;     Inheritance allows one class to acquire the properties and methods of another class.

Advantages:

Code reusability

Reduced code duplication

Easier maintenance



6\. **Polymorphism**

&#x20;    Polymorphism means "many forms." It allows the same method or operation to behave differently depending on the object.

Advantages:

Flexibility

Reusability

Simplified code



7\. **Method Overloading**

&#x20;    Method Overloading refers to using the same method name with different sets of parameters to perform different tasks.



8\. **Method Overriding**

&#x20;    Method Overriding occurs when a child class provides a different implementation of a method that is already defined in the parent class.

**Concept        	                Definition**
