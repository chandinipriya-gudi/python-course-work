###### **INPUT DECLARATION**



str- input()

int- int(input())

float- float(input())



list of str-input().split()

tuple of str-tuple(input().split())

set of str-set(input().split())



splitting string into list

The map() function applies a specified function to every item in an iterable (such as a list, tuple, or string) and returns a map object.

list of int - list(map(int,input().split()))

tuple of int - tuple(map(int,input().split()))

set of int - list(map(int,input().split()))



list of float - list(map(float,input().split()))

tuple of float - tuple(map(float,input().split()))

set of float - set(map(float,input().split()))



In Python, **eval()** is a built-in function that evaluates a string as a Python expression and returns the result.
